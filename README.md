# Usage

## Prerequisites

You must have Java Development Kit(JDK) 25 installed to compile and run TAI

```sh
git clone https://github.com/Yahm3/TAI.git
cd TAI
```

## Running the application

Run the `run.sh` script that handles the compilation and packaging

```sh
chmod +x run.sh
./run.sh
```
